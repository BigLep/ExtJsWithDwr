ExtJsWithDwr by BigLep

Tools for using Ext JS with DWR.

Ext JS makes creating beautiful web applications easy and fun. Direct Web Remoting (DWR) makes passing Java objects between the browser and Java web server a breeze. "ExtJsWithDwr" makes using Ext with DWR easier by providing a DWR implementation for some of Ext's abstractions (e.g., Ext.data.DataProxy).

Dependencies

ExtJs (http://extjs.com) DWR (http://directwebremoting.org/)

Install

This project assumes you are familiar with Ext JS and DWR. Learning how to use these integration tools is best done by looking at the examples.
